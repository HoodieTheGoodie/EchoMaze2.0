=========================

After uploading:

2. Click it to test the game
3. The game should load immediately in the embedded iframe
4. Try playing a bit to make sure everything works

If you see a blank screen:
- Check that you selected "index.html" as the index file
- Make sure "This file will be played in browser" is checked
- Try increasing viewport dimensions to 1024x768
