ABOUT THE GAME
--------------
Echo Maze is a strategic puzzle game where you navigate through challenging 
mazes while managing your abilities like shield and dash. Repair generators, 
avoid enemies, and find the exit as fast as you can!

Features:
• 10 Story Levels + Secret Boss Level
• Endless Mode with permanent upgrades
• Custom Level Builder
• Multiple skins and achievements
• Strategic gameplay with timing-based challenges


HOW TO PLAY
-----------
1. Extract all files from this ZIP archive to a folder on your computer
2. Open "index.html" in your web browser (Chrome, Firefox, Edge, Safari, etc.)
3. The game will start automatically!

IMPORTANT: Keep all files in the same folder structure. The game needs the 
"game" folder with all its contents to work properly.


CONTROLS
--------
• Arrow Keys or WASD - Move
• Space Bar - Shield (blocks damage and projectiles)
• Shift - Dash (move faster, consumes stamina)
• F - Use trap (when available)
• E - Interact with generators/terminals
• Escape - Pause

Mobile/Touch Controls:
• On-screen buttons appear automatically on touch devices


GAME MODES
----------
STORY MODE: Progress through 10 levels, unlock new mechanics
ENDLESS MODE: Survive infinite randomly generated rooms, earn permanent upgrades
LEVEL BUILDER: Create and play custom levels


TIPS
----
• Your shield can block projectiles - time it right!
• Dashing through walls when low on stamina damages you
• Each enemy type has unique behaviors - learn their patterns
• Generators must be repaired before you can exit
• Terminal puzzles unlock special abilities


OFFLINE PLAY
------------
This game is 100% offline! No internet connection required after download.
All your progress is saved locally in your browser.


TECHNICAL INFO
--------------
• Built with HTML5, JavaScript, and Canvas
• No installation or plugins required
• Works on Windows, Mac, Linux
• Compatible with modern web browsers
• Fully self-contained - no external dependencies

TROUBLESHOOTING
---------------
If the game doesn't load:
1. Make sure you extracted ALL files from the ZIP
2. Try a different web browser
3. Check that JavaScript is enabled in your browser
4. Ensure you're opening "index.html" (not a file from inside the game folder)


PRIVACY
-------
This game does not collect any data, track analytics, or require any personal 
information. Everything runs locally on your computer.
